How to Use

    Edit, delete, create files, configs, or anything else in Directories

    - Files created in "/conf" will be used as startup configurations, "snort" does not need to be specified in the configuration file

    - Example:"-c /usr/local/etc/snort/snort.lua -R /usr/local/etc/rules/local.rules -i eth0 -A alert_full -l /var/log/snort"
        - Saved as example.conf in "/usr/local/snort/conf/" with the inbuilt file explorer

    Start Snort and monitor terminal output in Snort

    - Snort is fully controlled here, all terminal output is shown to the right of the screen
